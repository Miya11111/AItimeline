# Color System

このプロジェクトの色管理システムのドキュメントです。

## 概要

すべての色はlight/darkテーマに対応しており、`useColors`フックを使用することで自動的にテーマに応じた色が適用されます。

## ファイル構成

- **[color.ts](color.ts)** - すべての色の定義（light/dark対応）
- **[theme.ts](theme.ts)** - 後方互換性のためのレガシーテーマ設定

## 基本的な使用方法

### useColors フック（推奨）

`useColors`を使うと、`colors.primary`のように書くだけで自動的にテーマに応じた色が適用されます。

```typescript
import { useColors } from '@/hooks/use-colors';

function MyComponent() {
  const colors = useColors();

  return (
    <View style={{
      backgroundColor: colors.primary,  // 自動的にlight/darkが選ばれる
      borderColor: colors.gray,
    }}>
      <Text style={{ color: colors.black }}>Hello</Text>
    </View>
  );
}
```

### StyleSheetでの使用

```typescript
import { useColors } from '@/hooks/use-colors';
import { StyleSheet } from 'react-native';

function MyComponent() {
  const colors = useColors();

  const styles = StyleSheet.create({
    container: {
      backgroundColor: colors.white,
      borderColor: colors.gray,
      borderWidth: 1,
    },
    text: {
      color: colors.black,
    },
    button: {
      backgroundColor: colors.primary,
    },
  });

  return <View style={styles.container}>...</View>;
}
```

## 利用可能な色

### ブランドカラー

- `primary` - プライマリーカラー
  - Light: `#48b3ffff`
  - Dark: `#fff`
- `caution` - 注意色
  - Light: `#f3e460ff`
  - Dark: `#ffd54f`
- `warning` - 警告色
  - Light: `#e63232ff`
  - Dark: `#e57373`

### グレースケール

- `gray` - グレー
  - Light: `#b4b4b4ff`
  - Dark: `#9BA1A6`

### 基本色

- `white` - 白
  - Light: `#fff`
  - Dark: `#151718` (ダークモードでは暗い色)
- `black` - 黒
  - Light: `#000`
  - Dark: `#fff` (ダークモードでは明るい色)

## 直接色定義を参照する場合

特定のテーマの色を直接参照したい場合は、color.tsから直接インポートできます。

```typescript
import { colors } from '@/constants/color';

// 特定のテーマの色を取得
const primaryLight = colors.primary.light;
const primaryDark = colors.primary.dark;

// 手動でテーマを切り替える
const colorScheme = useColorScheme() ?? 'light';
const primaryColor = colors.primary[colorScheme];
```

## ベストプラクティス

### ✅ 良い例

```typescript
// useColorsを使用してテーマ対応
const colors = useColors();
const backgroundColor = colors.primary;

// コンポーネント内でスタイル定義
function MyComponent() {
  const colors = useColors();
  const styles = StyleSheet.create({
    container: { backgroundColor: colors.white }
  });
  return <View style={styles.container} />;
}
```

### ❌ 悪い例

```typescript
// ハードコードされた色値
const backgroundColor = '#48b3ffff';

// テーマ非対応のグローバルスタイル
const styles = StyleSheet.create({
  container: { backgroundColor: '#fff' },
});

// useColorsを使わない直接参照
import { colors } from '@/constants/color';
const primary = colors.primary; // ❌ これはオブジェクト {light, dark} が返される
```

## 既存コードの移行

### 旧コード

```typescript
import { colors } from '@/constants/color';
const primary = colors.primary; // '#48b3ffff'
```

### 新コード

```typescript
import { useColors } from '@/hooks/use-colors';

function MyComponent() {
  const colors = useColors();
  const primary = colors.primary; // テーマに応じて自動選択
}
```

## 型安全性

すべての色はTypeScriptで完全に型付けされています。

```typescript
import { useColors } from '@/hooks/use-colors';

const colors = useColors();
const primary = colors.primary; // ✅ OK (string)
const invalid = colors.invalidKey; // ❌ コンパイルエラー
```
